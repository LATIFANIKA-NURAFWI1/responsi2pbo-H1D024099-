Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa fungsi keyword 'extends' dalam deklarasi class Java?
2. Jelaskan perbedaan antara Method Overloading (yang ada di latihan sebelumnya) dengan Method Overriding (yang ada di latihan ini)!
3. Apa kegunaan keyword 'super' di dalam constructor subclass (Manajer)?
4. Jika variabel di class Karyawan diubah access modifier-nya menjadi 'private', apa yang akan terjadi pada class Manajer? Bagaimana solusinya?
5. Mengapa kita perlu menggunakan Inheritance? Apa keuntungan utamanya dibandingkan membuat dua class yang terpisah sama sekali (misal class Karyawan dan class Manajer dibuat terpisah tanpa hubungan)?

Jawaban:
1. Extends digunakan untuk menerapkan konsep Inheritance/pewarisan. Jadi sebuah class (subclass/child class) mewarisi atribut dan method dari class lain (superclass/parent class).
2. => Overloading: membuat method dengan nama sama, tapi parameter berbeda dalam class yang sama. Untuk menangani berbagai jenis input
   => Overriding: menulis ulang method yang sudah ada di parent class ke subclass dengan nama dan parameter yang sama persis seperti di parent class. Untuk menyesuaikan method tersebut dengan subclassnya (seperti method tampilInfo pada Manajer)
3. Untuk memanggil constructor dari parent class (Karyawan)
4. Class Manajer akan error, karena variabel private tidak bisa diakses langsung oleh subclass (Karyawan). Solusinya: menggunakan method setter dan getter di class Karyawan agar Manajer bisa mengakses/mengubah data tersebut
5. => reusability: menghindari duplikasi kode, jadi kita tidak perlu menulis ulang kode nama dan gajiPokok di setiap jenis karyawan baru, cukup mewarisi dari Karyawan
   => maintenance: memudahkan maintenance jika ada perubahan pada Karyawan, cukup ubah di parent class, maka semua subclass akan otomatis terupdate
   => hierarki: kode menjadi lebih logis dan terstruktur
   Daripada membuat dua class terpisah, adanya inheritance memungkinkan kita untuk memanfaatkan kode yang sudah ada (reuseability) dan lebih mudah dalam meintenance
